# incrementr

## Commands
* Project setup:
  * `npm install`
* Compile and hot-reload for development:
  * `npm run serve`
* Lints and fixes files:
  * `npm run lint`
* Compiles, minifies, and deploys to github pages:
  * `npm run deploy`
